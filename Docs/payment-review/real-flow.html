本文件以測試帳號實際操作，逐頁擷取 FLESIM 網站真實畫面：選擇目的地 → 選擇 eSIM／實體 SIM 套餐與價格幣種 → 購物車 → 結帳（填寫 email／收件資料 + 信用卡支付）→ 支付結果 → 虛擬 eSIM 交付（QR）／實體 SIM 交付（物流）→ 訂單查詢 → 售後及退款入口。

其中「支付成功／失敗」頁為示意（mock）；「3D Secure」因尚未串接真實金流，附說明頁替代。其餘均為真站截圖。
步驟 1 — 選擇目的地國家/地區
首頁選擇國家；右上角可切換幣別（USD / TWD / HKD）。
步驟 2 · 3 — 選擇 eSIM／實體 SIM 套餐（流量·天數）與價格幣種
可切換 eSIM／SIM 卡分頁，逐一列出電信商方案與起價（幣別）。
步驟 — 購物車（價格及幣種展示）
購物車依 eSIM／SIM 分區顯示商品、方案、價格與數量、合計。
步驟 4 · 6 — 填寫 email／收件資料 + 信用卡支付
結帳頁填寫 email（eSIM）與收件地址（實體 SIM），並輸入信用卡卡號、到期日、CCV。
步驟 7 — 3D Secure 驗證
信用卡付款送出後，導向發卡銀行 3D Secure 進行持卡人身分驗證。
說明：本商戶目前尚未串接真實金流之 3D Secure（發卡行驗證）頁面，故暫無實際 3DS 頁面可供展示；信用卡付款頁（含 3DS 導向機制）如前頁所示。待金流正式串接完成後，將補送實際 3D Secure 驗證頁截圖。
步驟 8 — 支付成功（示意）
付款完成後於網站顯示付款成功與訂單編號，可查看訂單。
步驟 8b — 支付失敗（示意）
付款未成功時於網站顯示付款失敗，可返回選購重試。
交付（虛擬）— eSIM QR Code / 訂單詳情
訂單詳情展示 eSIM 安裝 QR、SM-DP+ 位址與啟用碼（並寄送至 email）。
交付（實體）— 實體 SIM 配送 / 物流資訊
實體 SIM 訂單展示收件資訊、物流商、追蹤號碼與配送狀態；下方含售後退款入口。
步驟 — 訂單查詢入口
會員中心「訂單查詢」列出所有訂單與狀態。
步驟 — 售後申請及退款入口
訂單詳情提供「申請售後 / 退款」入口，導向售後申請頁。
會員中心 / 帳號
會員可於會員中心管理訂單、卡片與帳號。
