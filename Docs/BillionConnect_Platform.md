# BillionConnect 平台結構說明

> 本文件記錄上游平台（BillionConnect）的頁面結構、功能說明與截圖觀察，
> 作為自建系統的設計參考。

---

## 整體佈局

```
┌─────────────────────────────────────────────────────┐
│  Logo   │  麵包屑導航              │ 餘額 │ 語言 │ 通知 │ 帳號 │
├─────────┼─────────────────────────────────────────────┤
│         │                                             │
│  側邊欄  │              主要內容區                      │
│  導航列  │                                             │
│         │                                             │
└─────────┴─────────────────────────────────────────────┘
```

- **頂部 Header**：顯示當前頁面路徑（麵包屑）、帳戶餘額（如 CNY 864.52）、語言切換（簡體中文）、通知鈴鐺、帳號頭像
- **左側邊欄**：固定導航，可收合，寬約 240px
- **主內容區**：依頁面切換

## 導航結構與功能說明

### 儀表板

- 首頁概覽
- 預計顯示：帳戶餘額、近期訂單統計、公告摘要

---

### eSIM 訂單

#### 創建訂單

分三個 Tab：

**非復充 eSIM**
- 篩選條件（依序聯動下拉）：
  1. 搜尋國家或地區
  2. 選擇天數
  3. 選擇流量
  4. 選擇 APN
  5. 搜尋套餐名稱或 ID（必填，示例：日本 3GB 7天、香港 澳門 300MB 7天）
- 套餐數量：`-` / 數字 / `+`
- 訂單金額顯示
- 更多選項（選填，展開）
- 批量導入按鈕（右上角）
- 確認創建按鈕（右下角）
- 對應 API：F001 → F002 → F003 → **F040**

**復充 eSIM**
- 針對已有 eSIM 卡進行套餐復充
- 對應 API：F052 → F002 → **F007**

**eSIM Air**
- eSIM Air 類型商品下單
- 對應 API：F001 → F002 → **F040**（type=250）

#### 創建充值訂單

**復充 eSIM**
- 輸入 ICCID → 查詢可充值套餐 → 建單
- 對應 API：F013 驗證 → F052 → **F007**

**eSIM Air**
- eSIM Air 充值
- 對應 API：**F007**

#### 創建第三方訂單

**非復充 eSIM / 復充 eSIM**
- 第三方渠道訂單建立
- 對應 API：**F040**

#### 訂單列表

- 查詢所有 eSIM 訂單狀態
- 對應 API：**F011**

---

### SIM 卡訂單

#### 創建訂單

分三個 Tab：

**充值**
- 輸入 ICCID → 驗證 → 選擇套餐 → 建單
- 對應 API：F013 → F002 → **F007**

**郵寄**
- 選擇商品 → 填寫收件地址 → 選擇物流公司 → 建單
- 對應 API：F002 → F005 → **F006**（含 express 物件）

**自提**
- 選擇商品 → 選擇自提點 → 建單
- 對應 API：F004 / F051 → **F006**（含 selfPickup 物件）

#### 創建第三方訂單

- 充值 / 郵寄 / 自提（同上，來源標記為第三方）

#### 訂單列表

- 查詢所有 SIM 卡訂單
- 對應 API：**F011**

---

### 套餐列表

#### eSIM 套餐列表

- 顯示所有 eSIM 類型商品（type: 230、250）
- 對應 API：**F002**（salesMethod 篩選）

#### SIM 套餐列表

**套餐商品**
- 流量套餐（type: 110、111）

**卡片商品**
- 含卡商品（type: 210、211、212 等）
- 對應 API：**F002**

#### 加速包套餐列表

- 所有加速包商品
- 對應 API：**F056**

---

### 運營管理

#### 套餐信息查詢

- 輸入 ICCID 查詢套餐使用狀況
- 對應 API：**F012** / **F046**

#### 售後列表

- 查詢、提交、修改、取消售後單
- 對應 API：**F017 / F018 / F019 / F020**

#### 實名認證狀態

- 查詢 ICCID 實名認證進度
- 對應 API：**F054**

#### ICCID 管理

| 子頁面 | 說明 | 對應 API |
|--------|------|----------|
| 有效期查詢 | 查詢卡片到期日 | F010 |
| 延期訂單查詢 | 查詢已延期的訂單 | — |
| 升級訂單查詢 | 查詢卡片升級紀錄 | — |
| 重置訂單查詢 | 查詢重置紀錄 | — |
| 自動充值查詢 | 查詢自動充值設定 | — |

---

### 帳單管理

#### 帳戶充值

- 查詢現有餘額、發起充值
- 對應 API：**F014**

#### 帳單明細

- 消費流水查詢（依時間區間）

#### 發票管理

**可開發票**
- 列出可申請開票的訂單

**開票歷史**
- 已開票紀錄查詢

---

### 報表

#### 採購對帳單

- 依時間區間統計採購金額與數量

#### 套餐銷售統計

- 各套餐商品銷售量、金額統計
- 可搭配 **F023**（日流量）做趨勢分析

---

### 公告

#### 業務公告

- 業務相關通知（如套餐下架、價格調整）

#### 系統公告

- 系統維護、接口版本更新通知

---

### 帳號管理

#### 帳號管理

- 新增/編輯子帳號

#### 角色管理

- 定義角色權限（如只讀、下單、管理員）

## UI 設計規格（從截圖推測）

- **主色**：藍紫色（#5B4FCF 左側選中項）、白色背景
- **字體**：系統字體，簡體中文
- **表單風格**：圓角下拉選單、灰色 placeholder
- **按鈕**：主操作按鈕（藍色，右下角）、批量操作（右上角白底灰邊）
- **側邊欄**：選中項背景色藍紫、子選單可展開/收合
